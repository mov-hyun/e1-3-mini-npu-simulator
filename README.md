## 2. 개념 정리

### 2.1 MAC 연산

MAC(Multiply and Accumulate) 연산은 두 값을 곱한 뒤, 그 결과를 계속 누적해서 더해 나가는 연산이다.  
즉, 하나의 연산 안에서 곱셈(Multiply)과 누적 덧셈(Accumulate)이 함께 이루어진다.

이 프로젝트에서는 입력 패턴과 필터의 같은 위치에 있는 값을 서로 곱하고, 그 결과를 모두 더해 하나의 점수를 만든다.  
이 점수는 입력 패턴이 특정 필터와 얼마나 유사한지를 판단하는 기준이 된다.

MAC 연산은 AI에서 매우 중요한 역할을 한다.  
AI 모델은 입력 데이터와 가중치를 반복해서 곱하고 더하는 방식으로 동작하며, 이미지 분류나 패턴 인식 같은 작업도 결국 이러한 연산의 반복으로 이루어진다.  
하나의 MAC 연산은 단순해 보이지만, 실제 AI 모델에서는 이 연산이 매우 큰 크기의 데이터에 대해 수없이 반복된다.  
따라서 MAC 연산을 얼마나 빠르게 처리할 수 있는지가 전체 AI 성능에 직접적인 영향을 주며, 이러한 이유로 대량의 MAC 연산을 병렬로 처리하는 NPU가 중요한 하드웨어로 사용된다.
