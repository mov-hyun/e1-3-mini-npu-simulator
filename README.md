## 1. 프로젝트 개요

### 1.1 프로젝트 소개

- Python 기반 `Mini NPU Simulator` 구현
- CLI 환경에서 패턴과 필터를 비교하는 콘솔 애플리케이션 개발
- 반복문 기반 MAC(Multiply-Accumulate) 연산 직접 구현
- `data.json` 기반 패턴 분석 및 판정 결과 출력
- 크기별 연산 시간 측정과 결과 리포트 작성
- `Python 3.8+`, 표준 라이브러리(`json`, `time`) 사용
- 외부 라이브러리 없이 구현
